【LeetCode】121. 买卖股票的最佳时机

解题思路

最大子序列和求解
思路参考自LeetCode的某位同学题解
对其中的将两者之差转换为最大连续子序列和的更形象的解释如下：
对于a[i]-a[j]可展开成如下形式
a[i]-a[j]
=a[i]-a[i-1]+a[i-1]-a[i-2]+…+a[j+1]-a[j]
=diff[i]+diff[i-1]+…+diff[j+1]
举例：假设输入为[7,1,5,3,6,4]的序列，输出为5，
则我们知道a[4]-a[1]为最大，等于5
按上述思路可知：
a[4]-a[1]=diff[4]+diff[3]+diff[2]
而diff数组为[-6,4,-2,3,-2]，注意假设diff数组从0开始，则对应于上文的diff[1]对应上文的diff[2],依次类推
diff[2]=4,diff[3]=-2,diff[4]=3
因此a[4]-a[1]=4-2+3=5

代码

class Solution {
public:
	int maxProfit(vector<int>& prices) {
		if (prices.size() <= 1) return 0;//prices.size()只有1个或0个元素，直接返回0
		vector<int> diff(prices.size() - 1);//存储两两的差值
		vector<int> dp(prices.size() - 1);//dp[i]表示以diff[i]作为末尾的连续序列的最大和
		for (int i=0;i<prices.size()-1;i++)
		{
			diff[i] = prices[i + 1] - prices[i];//计算diff
		}
		//边界
		dp[0] = max(0,diff[0]);//注意这里但对初始状态要进行一个比较，如果diff[0]比0小，则要取0
		//状态转移方程
		for (int i=1;i<diff.size();i++)
		{
			dp[i] = max(0, dp[i - 1] + diff[i]);
		}
		//输出连续子序列的最大值
		vector<int>::iterator myMax = max_element(dp.begin(), dp.end());//求出dp中的最大值
		return *myMax;
	}
};
